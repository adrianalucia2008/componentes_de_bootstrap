Accordion

El componente “Accordion” de Bootstrap sirve para mostrar contenido dividido en secciones plegables (expandibles/colapsables) dentro de una misma área, permitiendo que el usuario vea solo una parte de la información a la vez

This is the first item’s accordion body. It is shown by default, until the collapse plugin adds the appropriate classes that we use to style each element. These classes control the overall appearance, as well as the showing and hiding via CSS transitions. You can modify any of this with custom CSS or overriding our default variables. It’s also worth noting that just about any HTML can go within the .accordion-body, though the transition does limit overflow.

Buttons

El componente “Button” de Bootstrap sirve para crear botones interactivos y estilizados que pueden desencadenar acciones, enviar formularios o navegar a otras páginas dentro de una aplicación web.

Button group

El componente “Button Group” de Bootstrap sirve para agrupar varios botones relacionados en una sola línea o bloque, permitiendo una organización más compacta y coherente de las acciones disponibles para el usuario.

Cards

El componente “Card” de Bootstrap sirve para mostrar contenido agrupado en un contenedor estilizado, que puede incluir imágenes, texto, enlaces y otros elementos, proporcionando una forma visualmente atractiva y organizada de presentar información.

Carousel

El componente “Carousel” de Bootstrap sirve para mostrar una serie de elementos (como imágenes o contenido) de manera deslizante y cíclica, permitiendo a los usuarios navegar a través de ellos mediante controles o indicadores.

Close button

El componente “Close Button” de Bootstrap sirve para proporcionar un botón estilizado que permite a los usuarios cerrar o descartar elementos emergentes, modales, alertas u otros componentes interactivos en una interfaz web.

Collapse

El componente “Collapse” de Bootstrap sirve para mostrar u ocultar contenido adicional de manera dinámica, permitiendo a los usuarios expandir o contraer secciones de información según sus necesidades.

Modals

El componente “Modal” de Bootstrap sirve para mostrar ventanas emergentes superpuestas en la interfaz de usuario, que pueden contener contenido adicional, formularios o mensajes importantes, permitiendo a los usuarios interactuar con ellos sin salir de la página actual.

Placeholders

El componente “Placeholder” de Bootstrap sirve para mostrar elementos de carga visuales mientras se espera que el contenido real se cargue, proporcionando una experiencia de usuario más fluida y atractiva durante los tiempos de espera.

Popovers

El componente “Popover” de Bootstrap sirve para mostrar cuadros de diálogo emergentes que proporcionan información adicional o contextual cuando el usuario interactúa con un elemento específico, como hacer clic o pasar el cursor sobre él.

Progress

El componente “Progress” de Bootstrap sirve para mostrar visualmente el progreso de una tarea o proceso en curso, proporcionando una barra que se llena gradualmente a medida que avanza la operación.

Scrollspy

El componente “Scrollspy” de Bootstrap sirve para actualizar automáticamente los enlaces de navegación en función de la posición de desplazamiento del usuario en la página, resaltando la sección activa en la barra de navegación a medida que se desplaza por el contenido.

Tooltip

El componente “Tooltip” de Bootstrap sirve para mostrar pequeñas ventanas emergentes con información adicional cuando el usuario pasa el cursor sobre un elemento o hace clic en él, proporcionando contexto o detalles sin ocupar espacio permanente en la interfaz.
